##  Installation

- Clone This Front End Repo

```
git clone https://github.com/CandaMuammal/FE-Telegram

```

- Go To Folder Repo

```
cd FE-Telegram
```

- Install Module

```
npm install
```

- Type ` npm start` To Start Website
